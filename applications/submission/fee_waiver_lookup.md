# Fee Waiver Lookup

**Purpose:** Map each target program to its fee waiver options. Apply for waivers EARLY — most waiver windows close 2-4 weeks before the application deadline. Total fee budget if no waivers: ~$1,600-3,200 across 22+ programs.

---

## Universal waiver paths (try these first)

These cross-program waiver mechanisms can apply to multiple US schools:

### 1. NSF Graduate Research Fellowship Program (NSF GRFP) applicant
- **Eligibility:** US citizen / permanent resident; in your first or second year of graduate study, or applying to start
- **Auto-waivers at:** Stanford, MIT, Berkeley, Princeton, CMU (some others — verify per school)
- **How:** Submit NSF GRFP application by mid-October 2027 deadline; provide the GRFP application ID on each school's fee waiver form
- **Cost:** Free
- **Timeline:** NSF GRFP application opens August 2027, deadline mid-Oct 2027; school-by-school auto-waiver activates once you submit GRFP

### 2. GEM Fellowship (National GEM Consortium)
- **Eligibility:** US citizen / permanent resident; underrepresented in STEM
- **Auto-waivers at:** Many GEM member universities (full list at https://www.gemfellowship.org/)
- **How:** Submit GEM application separately; use member institutions
- **Cost:** Free
- **Timeline:** GEM deadline typically mid-November 2027

### 3. Hertz Foundation Fellowship applicant
- **Eligibility:** US citizen / permanent resident; applied sciences focus
- **Auto-waivers at:** Some Hertz fellowship target institutions (verify)
- **Timeline:** Hertz application typically due Oct 2027

### 4. FreeApp.io aggregator
- **Eligibility:** First-generation college student OR financial hardship
- **How:** Single application at https://www.freeapp.io/; sends to multiple participating schools
- **Status:** ~20 participating schools as of 2026; verify which target schools accept FreeApp for 2027-2028

### 5. NSBE / SHPE / SACNAS / oSTEM conference-recruiting codes
- **Eligibility:** Attend the conference (in-person or virtual)
- **How:** Many CS programs distribute fee-waiver codes at their recruiting tables. Visit booths and ask explicitly: "Are you offering fee-waiver codes for the 2028 application cycle?"

## Recommended fee-waiver application order

1. **August 2027:** Apply for NSF GRFP (if eligible). This is the highest-leverage single action.
2. **September 2027:** Apply for GEM Fellowship (if eligible).
3. **September 2027:** Apply for Hertz Foundation (if eligible).
4. **Early October 2027:** Submit FreeApp.io application (universal hardship/first-gen).
5. **Mid October 2027:** Submit university-system need-based waivers (UC waiver covers Berkeley + UCLA; Stanford has its own; UMass via CICS).
6. **Late October 2027:** Per-program need-based waivers for any program not covered by the above.
